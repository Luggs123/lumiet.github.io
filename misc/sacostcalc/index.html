How to use

First, you need to go to the Market to find the current prices for Delta Vouchers. Select the quantity you're viewing in the drop down and enter the recent sale price in Credits. This will calculate the DP price for the next section.
Then, select from the table the special that you want to know the price of. Once you click that table cell, it will calculate the price of that special.

Important note: These prices should only be used as a baseline and not everyone will want to sell based on DP price. This is meant to be a tool to help people figure out how to price their specials.
